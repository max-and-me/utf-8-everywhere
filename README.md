## Getting started

To clone and build the project open a command prompt and proceed as follows:

### Clone and Build

```sh
git clone https://github.com/max-and-me/utf-8-everywhere.git
mkdir build
cd buid
cmake ../utf-8-everywhere
cmake --build .
```
